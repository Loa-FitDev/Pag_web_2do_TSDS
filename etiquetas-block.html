Introducción a Bootstrap

Bootstrap es un framework de código abierto muy popular para desarrollar sitios web responsivos y móviles. Proporciona una colección de componentes predefinidos que facilitan el desarrollo web rápido.

Con Bootstrap, puedes crear diseños profesionales sin necesidad de escribir mucho CSS personalizado. El framework utiliza un sistema de grid de 12 columnas que se adapta a diferentes tamaños de pantalla.

Barra Lateral - <aside>

Dato Interesante

Bootstrap fue creado por Mark Otto y Jacob Thornton en Twitter en 2011. Desde entonces, se ha convertido en uno de los frameworks más populares para el desarrollo web.
