#  Ingeniería II - Tecnicatura en Desarrollo de Software

##  Información del Proyecto

Este repositorio corresponde a la materia **Ingeniería II** de **2° año** de la **Tecnicatura Superior en Desarrollo de Software** del **Instituto Superior de Profesorado N°20**.

Aquí se desarrollarán trabajos prácticos, ejercicios y proyectos relacionados con el desarrollo de aplicaciones web.

---

##  Tecnologías Utilizadas

Durante la cursada se trabajará con las siguientes tecnologías:

-  **HTML5** – Estructura de páginas web  
-  **CSS3** – Estilos y diseño visual  
-  **JavaScript** – Interactividad y lógica del lado del cliente  
-  **Bootstrap** – Framework para diseño responsivo y componentes UI  

---

##  Contenido del Repositorio

Este repositorio incluirá:

- Trabajos prácticos individuales y grupales  
- Ejercicios de práctica  
- Proyectos web  
- Apuntes y ejemplos de código  

---

##  Objetivos

- Comprender los fundamentos del desarrollo web  
- Aplicar buenas prácticas de programación  
- Desarrollar interfaces modernas y responsivas  
- Integrar diferentes tecnologías frontend  
- Trabajar con control de versiones usando Git y GitHub
